# Matteo

  I build agentic AI systems that combine LLMs, retrieval, memory, multimodal document processing, and automation infrastructure.

  My work focuses on practical AI applications: systems that can read, search, reason over private data, use tools, and run reliably on self-
  hosted and cloud infrastructure.

  ## Current Focus

  - Agentic RAG and document intelligence
  - Multimodal OCR and metadata extraction
  - Long-term memory systems for AI agents
  - Self-hosted AI infrastructure and model serving
  - Workflow automation with LLM-powered tools

  ## Featured Projects

  ### docker-paperless-ai
  A platform for AI-assisted document processing on top of Paperless-ngx.
  It combines agentic RAG, multimodal OCR, metadata extraction, vector search, and self-hosted models to automate document understanding over
  private archives.

  ### camillo
  A memory backbone for AI agents.
  It explores durable memory, hybrid recall, associative links, forgetting, and memory reconciliation for long-running agentic systems.

  ### crabberbot
  A Telegram bot that downloads media from websites and processes it with AI.
  It connects external content ingestion, media handling, automation, and AI-assisted analysis.

  ### fish-speech
  Experiments with fine-tuning and custom layers for open-source text-to-speech models.
  This project reflects my interest in adapting foundation models for specialized use cases.
